## ඒකකය 1 හා 2: මිනුම් හා යාන්ත්‍ර විද්‍යාව

### 1. මිනුම් (Measurements)

- **SI මූලික ඒකක**: දිග (m), ස්කන්ධය (kg), කාලය (s), ධාරා ප්‍රබලතාව (A), උෂ්ණත්වය (K), පදාර්ථ ප්‍රමාණය (mol), දීප්ත තීව්‍රතාව (cd)
- **ව්‍යුත්පන්න ඒකක**: m/s, N, J, W, V, Ω ආදිය
- **මාන සූත්‍ර**: [M], [L], [T] වලින් ප්‍රකාශ කිරීම
- **දෝෂ**: නිරපේක්ෂ දෝෂය, සාපේක්ෂ දෝෂය, ප්‍රතිශත දෝෂය
- **සැලකිය යුතු සංඛ්‍යා**: මිනුම්වල නිරවද්‍යතාව පෙන්වන ඉලක්කම් ගණන (Sig Fig)
- **මිනුම් උපකරණ**: Vernier caliper (±0.01mm), Micrometer screw gauge (±0.01mm), Travelling microscope

### 2. චලිතය (Motion)

- **චලිත විශ්ලේෂණය**: v = u + at, s = ut + ½at², v² = u² + 2as
- **චලිත ප්‍රස්තාර**: displacement-time, velocity-time, acceleration-time
- **චාලක සමීකරණ** (constant acceleration සඳහා පමණි)
- **ප්‍රක්ෂේප චලිතය**: වක්‍ර ගමන් මාර්ගය, තිරස් පරාසය R = u²sin2θ/g, උපරිම උස h = u²sin²θ/2g
- **සරල අනුවර්තීය චලිතය (SHM)**: x = A sin(ωt+φ), v = ω√(A²-x²), a = -ω²x, T = 2π√(m/k)

### 3. බලය හා චාලක (Force & Dynamics)

- **නිව්ටන් නියම**:
  - පළමු නියමය: අවස්ථිති බව
  - දෙවන නියමය: F = ma
  - තෙවන නියමය: ක්‍රියා-ප්‍රතික්‍රියා
- **ගම්‍යතාව**: p = mv
- **ආවේගය**: I = Ft = Δp
- **ඝර්ෂණය**: f ≤ μN (ස්ථිතික), f = μₖN (ගතික)
- **ඇදීම් බලය (Drag)**: ප්‍රවේගයට සමානුපාතික, අවසාන ප්‍රවේගය

### 4. වෘත්ත චලිතය (Circular Motion)

- **ω = v/r**, T = 2π/ω, f = 1/T
- **කේන්ද්‍රාභිමුඛ ත්වරණය**: a = v²/r = ω²r
- **කේන්ද්‍රාභිමුඛ බලය**: F = mv²/r = mω²r
- **කේන්ද්‍රාපසාරී බලය**: මෙය අථික බලයක් නොව, අවස්ථිති බව නිසා ඇතිවන හැඟීමකි

### 5. කාර්යය, ශක්තිය හා ක්ෂමතාව (Work, Energy & Power)

- **කාර්යය**: W = Fd cosθ (ස්ථිර බලයක් සඳහා)
- **චාලක ශක්තිය**: Eₖ = ½mv²
- **විභව ශක්තිය**: Eₚ = mgh (ගුරුත්වාකර්ෂණ)
- **ශක්ති සංස්ථිති නියමය**: ශක්තිය නිර්මාණය හෝ විනාශ නොකළ හැක
- **ක්ෂමතාව**: P = W/t = Fv
- **ක්ෂමතාව (භ්‍රමණ)**: P = τω

### 6. ඝෘජු රේඛීය චලිතය (Rectilinear Motion) & භ්‍රමණ චලිතය (Rotational Motion)

- **භ්‍රමණ චාලක**: ω = ω₀ + αt, θ = ω₀t + ½αt², ω² = ω₀² + 2αθ
- **අවස්ථිති ඝූර්ණය (Moment of Inertia)**: I = Σmr²
- **ව්‍යාවර්තය (Torque)**: τ = Iα
- **භ්‍රමණ ගම්‍යතාව**: L = Iω
- **භ්‍රමණ ගම්‍යතා සංස්ථිතිය**: බාහිර ව්‍යාවර්තයක් නොමැති විට L නියතයි
- **භ්‍රමණ චාලක ශක්තිය**: Eₖ = ½Iω²

### 7. සමතුලිතතාව (Equilibrium)

- **සමතුලිතතා කොන්දේසි**:
  - ΣF = 0 (බල සමතුලිතතාව)
  - Στ = 0 (ව්‍යාවර්ත සමතුලිතතාව)
- **බල යුග්මය**: එකම විශාලත්වයේ ප්‍රතිවිරුද්ධ දිශාවල සමාන්තර බල දෙකක්
- **ගුරුත්ව කේන්ද්‍රය**: වස්තුවක බර ක්‍රියා කරන ලක්ෂ්‍යය

## ඒකකය 3: දෝලන හා තරංග / තාප භෞතික විද්‍යාව

### 1. දෝලන (Oscillations)

- **SHM**: F ∝ -x, T = 2π√(l/g) (සරල අවලම්බය), T = 2π√(m/k) (ස්ප්‍රිං)
- **SHM ශක්තිය**: E = ½kA² (Eₖ + Eₚ = නියත)
- **අධික තෙතමනය සහිත දෝලන (Damped oscillations)**: ක්‍රමයෙන් විස්තාරය අඩුවේ
- **ප්‍රණෝදිත දෝලන (Forced oscillations)**: බාහිර බලයක් මගින් පවත්වා ගනී
- **අනුනාදය (Resonance)**: ප්‍රණෝදිත සංඛ්‍යාතය ස්වභාවික සංඛ්‍යාතයට සමාන වූ විට විස්තාරය උපරිම වීම

### 2. තරංග (Waves)

- **තරංග වර්ග**: තීර්යක් (transverse), අන්වායාම (longitudinal)
- **තරංග ගුණාංග**: v = fλ, T = 1/f
- **අනුකලනය (Superposition)**: තරංග දෙකක් හමු වූ විට විස්තාර එකතු වීම
- **නිරෝධනය (Interference)**: Young's double slit (λ = ax/D)
- **වර්තනය (Diffraction)**: One-slit, diffraction grating (dsinθ = nλ)
- **ධ්වනි තරංග**: ශබ්දය, ඩොප්ලර් ආචරණය (f' = f(v±u₀)/(v±uₛ))
- **ජංගම තරංග (Standing waves)**: නූලක ජංගම තරංග, fₙ = n/2L × √(T/μ)

### 3. තාප ගති විද්‍යාව (Thermodynamics)

- **තාප ධාරිතාව**: C = mc (විශිෂ්ට තාප ධාරිතාව c)
- **ගුප්ත තාපය**: Q = mL
- **වායු නියම**:
  - බොයිල් නියමය: PV = නියත (ස්ථිර T)
  - චාල්ස් නියමය: V/T = නියත (ස්ථිර P)
  - පීඩන නියමය: P/T = නියත (ස්ථිර V)
  - සර්ව වායු සමීකරණය: PV = nRT
- **තාප ගති විද්‍යාවේ පළමු නියමය**: ΔU = Q + W (Q = තාපය, W = කාර්යය, ΔU = අභ්‍යන්තර ශක්ති විපර්යාස)
- **තාප ගති විද්‍යාවේ දෙවන නියමය**: එන්ට්‍රොපිය වැඩිවීම
- **තාප සම්ප්‍රේෂණය**: සන්නයනය (Conduction), සංවහනය (Convection), විකිරණය (Radiation)
- **ස්ටීවන්-බෝල්ට්ස්මාන් නියමය**: P = εσAT⁴

## ඒකකය 4: ප්‍රකාශ විද්‍යාව / පදාර්ථයේ ගුණ

### 1. ප්‍රකාශ විද්‍යාව (Optics)

- **පරාවර්තන නියම**: θᵢ = θᵣ, ආපතන කිරණ, පරාවර්තිත කිරණ, සාමාන්‍ය රේඛාව එකම තලයේ
- **වර්තන නියම (Snell's law)**: n₁sinθ₁ = n₂sinθ₂, n = c/v
- **පූර්ණ අභ්‍යන්තර පරාවර්තනය (TIR)**: sinθc = n₂/n₁
- **තන්තු ප්‍රකාශ විද්‍යාව**: TIR මගින් ෆයිබර් ඔප්ටික් කේබල්වල ආලෝකය සම්ප්‍රේෂණය
- **තුනී කාච (Thin lenses)**: 1/f = 1/u + 1/v (කාච සූත්‍රය)
- **කාච නිෂ්පාදක සූත්‍රය**: 1/f = (n-1)(1/R₁ - 1/R₂)
- **ප්‍රස්තාර**: ප්‍රභා ගුණය (Magnification) m = v/u, බලය P = 1/f (D)
- **ධ්‍රැවීකරණය (Polarization)**: තීර්යක් තරංග පමණක් ධ්‍රැවීකරණය විය හැක, Malus නියමය I = I₀cos²θ

### 2. ප්‍රත්‍යාස්ථතාව (Elasticity)

- **Stress = F/A** (ආතතිය), **Strain = Δl/l** (වික්‍රියාව)
- **Young's modulus**: E = stress/strain
- **Hooke's නියමය**: F = kx (k = දැඩිතා නියතය)
- **ප්‍රත්‍යාස්ථ ශක්තිය**: E = ½kx² = ½F²/k

### 3. තරල යාන්ත්‍ර විද්‍යාව (Fluid Mechanics)

- **පීඩනය**: P = F/A, P = ρgh (ද්‍රව තීරුව)
- **Pascal නියමය**: සංවෘත තරලයක පීඩනය සෑම දිශාවකටම සමානව සම්ප්‍රේෂණය වේ
- **ආකිමිඩීස් මූලධර්මය**: ප්ලවන බලය = විස්ථාපිත තරලයේ බර
- **පෘෂ්ඨික ආතතිය (Surface Tension)**: γ = F/l, γ = hρgr/2 (කේශාකර්ෂණය)
- **දුස්ස්‍රාවිතාව (Viscosity)**: η, ස්ටෝක්ස් නියමය F = 6πηrv
- **බර්නූලි සමීකරණය**: P + ½ρv² + ρgh = නියත
- **ප්‍රවේග සම්බන්ධතාව**: A₁v₁ = A₂v₂ (අඛණ්ඩතා සමීකරණය)

## ඒකකය 5: ගුරුත්වාකර්ෂණ ක්ෂේත්‍ර (Gravitational Fields)

- **නිව්ටන්ගේ විශ්ව ගුරුත්වාකර්ෂණ නියමය**: F = GMm/r²
- **ගුරුත්වාකර්ෂණ ක්ෂේත්‍ර ප්‍රබලතාව**: g = F/m = GM/r²
- **ගුරුත්වාකර්ෂණ විභවය (Gravitational Potential)**: V = -GM/r (r → ∞ දී V = 0)
- **විභව ශක්තිය**: U = -GMm/r
- **ගැලවීමේ/මිදීමේ ප්‍රවේගය (Escape velocity)**: vₑ = √(2GM/R) = √(2gR)
- **කක්ෂීය ප්‍රවේගය (Orbital velocity)**: vₒ = √(GM/r)
- **චන්ද්‍රිකා චලිතය**: T² ∝ r³ (Kepler තුන්වන නියමය)
- **Kepler නියම**:
  1. ඉලිප්සාකාර කක්ෂ (සූර්යයා නාභියේ)
  2. සමාන කාලවලදී සමාන ක්ෂේත්‍ර ප්‍රදේශ
  3. T² ∝ a³ (a = අර්ධ-ප්‍රධාන අක්ෂය)
- **භූ ස්ථාවර කක්ෂ**: T = 24h, r ≈ 42,000km, එක් ස්ථානයක සිට නිශ්චලව පෙනේ

## ඒකකය 6: ස්ථිති විද්‍යුත් ක්ෂේත්‍ර (Electrostatic Fields)

- **කූලෝම් නියමය**: F = kq₁q₂/r² = q₁q₂/4πε₀r²
- **විද්‍යුත් ක්ෂේත්‍ර ප්‍රබලතාව**: E = F/q = kQ/r²
- **විද්‍යුත් විභවය**: V = kQ/r
- **විභව අන්තරය**: ΔV = V₂ - V₁ = -∫E·dr
- **සමාන්තර තහඩු ධාරිත්‍රකය**: E = V/d, σ = Q/A = ε₀E
- **ගවුස් නියමය**: Φ = ∮E·dA = Qₑₙ𝒸/ε₀
- **ධාරිතාව (Capacitance)**: C = Q/V
- **ධාරිත්‍රක ශක්තිය**: U = ½QV = ½CV² = ½Q²/C
- **පාරා විද්‍යුත් ද්‍රව්‍ය (Dielectrics)**: C = κC₀ (κ = පාරා විද්‍යුත් නියතය)
- **ධාරිත්‍රක සංයෝජන**:
  - ශ්‍රේණි: 1/Cₜ = 1/C₁ + 1/C₂ + ...
  - සමාන්තර: Cₜ = C₁ + C₂ + ...

## ඒකකය 7: ධාරා විද්‍යුතය (Current Electricity)

- **ධාරාව**: I = Q/t, I = nAve (v = ඉසිලුම් ප්‍රවේගය)
- **ඕම් නියමය**: V = IR
- **ප්‍රතිරෝධය**: R = ρl/A (ρ = ප්‍රතිරෝධතාව)
- **ප්‍රතිරෝධක සංයෝජන**:
  - ශ්‍රේණි: Rₜ = R₁ + R₂ + ...
  - සමාන්තර: 1/Rₜ = 1/R₁ + 1/R₂ + ...
- **EMF හා අභ්‍යන්තර ප්‍රතිරෝධය**: V = E - Ir (E = emf, r = අභ්‍යන්තර ප්‍රතිරෝධය)
- **කර්කෝෆ් නියම**:
  - KCL: ΣIᵢₙ = ΣIₒᵤₜ (සන්ධියක දී)
  - KVL: ΣV = 0 (සංවෘත පරිපථයක)
- **විභවමානය (Potentiometer)**: emf සැසඳීම, E₁/E₂ = l₁/l₂
- **මීටර් සේතුව (Meter Bridge)**: Wheatstone සේතුව Rₓ/R₂ = R₁/R₃
- **විද්‍යුත් ශක්තිය හා බලය**: P = VI = I²R = V²/R
- **කිලෝවොට් පැය (kWh)**: විදුලි බිල්පත් ගණනයට

## ඒකකය 8: චුම්බක ක්ෂේත්‍ර (Magnetic Fields)

- **චලනය වන ආරෝපණ මත බලය**: F = qvBsinθ (Lorentz බලය)
- **ධාරාවක් ගෙන යන සන්නායක මත බලය**: F = BILsinθ
- **Biot-Savart නියමය**: dB = μ₀Idℓsinθ/4πr²
- **සෘජු ධාරා සන්නායකයක චුම්බක ක්ෂේත්‍රය**: B = μ₀I/2πr
- **සොලෙනොයිඩයක චුම්බක ක්ෂේත්‍රය**: B = μ₀nI (n = ඒකක දිගට වට ගණන)
- **විද්‍යුත් චුම්බක ප්‍රේරණය**:
  - ෆැරඩේ නියමය: E = -dΦ/dt
  - ලෙන්ස් නියමය: ප්‍රේරිත emf විපර්යාසයට විරුද්ධ වේ
- **ස්ව ප්‍රේරණතාව (Self-inductance)**: E = -L(dI/dt), U = ½LI²
- **අන්යෝන්ය ප්‍රේරණතාව (Mutual inductance)**: E₂ = -M(dI₁/dt)
- **ප්‍රත්‍යාවර්ත ධාරා (AC)**:
  - V = V₀sin(ωt), I = I₀sin(ωt-φ)
  - RMS: Vᵣₘₛ = V₀/√2, Iᵣₘₛ = I₀/√2
  - ප්‍රතිබාධනය (Impedance): Z = √(R² + (XL-XC)²)
  - ප්‍රේරක ප්‍රතිබාධනය: XL = ωL
  - ධාරිත්‍රක ප්‍රතිබාධනය: XC = 1/ωC
  - බල සාධකය: cosφ = R/Z
- **ට්‍රාන්ස්ෆෝමරය**: Vₛ/Vₚ = Nₛ/Nₚ, IₛVₛ = IₚVₚ (පරමාදර්ශී)
- **AC ජනකය (Generator)**: E = NBAωsin(ωt)

## ඒකකය 9: ඉලෙක්ට්‍රොනික විද්‍යාව (Electronics)

### 1. අර්ධ සන්නායක (Semiconductors)

- **ස්වභාවික අර්ධ සන්නායක**: Si, Ge (4 සංයුජ ඉලෙක්ට්‍රෝන)
- **n-type**: පෙන්ටවේලන්ට් අපද්‍රව්‍ය (P, As, Sb) - අතිරික්ත ඉලෙක්ට්‍රෝන
- **p-type**: ට්‍රයිවේලන්ට් අපද්‍රව්‍ය (B, Al, In) - ශූන්‍ය (hole) අතිරික්තය
- **සංයුජ පට්ටි ප්‍රාන්තරය (Band gap)**: සන්නායක < 0.1eV, අර්ධ සන්නායක ~1eV, පාර විද්‍යුත් > 5eV

### 2. p-n සන්ධි ඩයෝඩය (p-n Junction Diode)

- **ඉදිරි අගතිය (Forward bias)**: p-side (+) → සන්නයනය
- **ප්‍රතිලෝම අගතිය (Reverse bias)**: n-side (+) → සන්නයනය නැත
- **බිඳ වැටීම් වෝල්ටීයතාව (Breakdown voltage)**: Zener effect, Avalanche effect
- **VI ලක්ෂණ**: ඉදිරි 0.7V (Si), 0.3V (Ge)
- **ඩයෝඩ භාවිත**: සුළු ධාරා නිවැරදි කිරීම (Rectification) - half-wave, full-wave

### 3. ට්‍රාන්සිස්ටරය (Transistor)

- **BJT**: npn, pnp වර්ග
- **අග්‍ර තුන**: Emitter (E), Base (B), Collector (C)
- **විස්තාරණය**: β = IC/IB (β ~50-300)
- **කාරක මාද**: active, cutoff, saturation
- **Transistor ලාභය**: Aᵥ = -RC/RE (common emitter)
- **තර්කන ද්වාර (Logic Gates)**: AND, OR, NOT, NAND, NOR, XOR, XNOR
- **ද්වාර සත්‍ය වගු**: මතක තබා ගත යුතු ආදාන-ප්‍රතිදාන සම්බන්ධතා

## ඒකකය 10: පදාර්ථයේ හා විකිරණවල ගුණ

### 1. පදාර්ථයේ ගුණ (Properties of Matter)

- **ප්‍රත්‍යාස්ථතාව**: Young's modulus (Y), Shear modulus (G), Bulk modulus (K)
- **Poisson අනුපාතය**: σ = -εₜᵣₐₙₛ/εₐₓᵢₐₗ
- **දුස්ස්‍රාවිතාව**: Newtonian, Non-Newtonian තරල
- **Poiseuille සමීකරණය**: Q = πΔPr⁴/8ηl (කේශනාලිකා ප්‍රවාහය)
- **Terminal velocity**: vₜ = 2r²(ρ-ρ₀)g/9η (Stokes නියමය)
- **පෘෂ්ඨික ආතතිය**: නාලිකා ඉහළ නැගීම h = 2γcosθ/ρgr
- **Bernoulli**: P₁ + ½ρv₁² = P₂ + ½ρv₂² (තිරස් වූ විට)

### 2. ප්‍රකාශ විද්‍යුත් ආචරණය (Photoelectric Effect)

- **Einstein සමීකරණය**: hf = Φ + Eₖ(max), Eₖ(max) = hf - Φ
- **Φ = කාර්ය ශ්‍රිතය (Work function)**: ලෝහයෙන් ඉලෙක්ට්‍රෝනයක් ඉවත් කිරීමට අවශ්‍ය අවම ශක්තිය
- **එළිබිම් සංඛ්‍යාතය (Threshold frequency)**: f₀ = Φ/h
- **නවත්වන විභවය (Stopping potential)**: Vₛ = Eₖ(max)/e
- **ප්‍රභා ප්‍රවාහය**: තීව්‍රතාවට සමානුපාතික (සංතෘප්ත ධාරාව)
- **ඡායා විද්‍යුත් සෛල (Photo cells)**: ආලෝක සංවේදක, සූර්ය කෝෂ

### 3. X-කිරණ (X-Rays)

- **නිෂ්පාදනය**: අධි වේග ඉලෙක්ට්‍රෝන ලෝහ ඉලක්කයක ගැටීමෙන්
- **Bragg නියමය**: 2dsinθ = nλ (ස්ඵටික විවර්තනය)
- **X-ray වර්ණාවලිය**: අඛණ්ඩ (Bremsstrahlung) + ලාක්ෂණික
- **Moseley නියමය**: √f ∝ Z

### 4. ප්‍රතිදීප්තතාව (Fluorescence) හා ෆොස්ෆරීසන්ස්

- **Fluorescence**: 10⁻⁸s ඇතුළත විමෝචනය
- **Phosphorescence**: ප්‍රමාද විමෝචනය (පැය ගණනක්)

## ඒකකය 11: නූතන භෞතික විද්‍යාව / න්‍යෂ්ටික භෞතික විද්‍යාව

### 1. න්‍යෂ්ටිය (Nucleus)

- **න්‍යෂ්ටික සංයුතිය**: ප්‍රෝටෝන (Z) + නියුට්‍රෝන (N), A = Z + N (ස්කන්ධ අංකය)
- **සමස්ථානික (Isotopes)**: එකම Z, විවිධ N
- **න්‍යෂ්ටික බන්ධන ශක්තිය**: ΔE = Δmc² (E = mc²)
- **ස්කන්ධ දෝෂය**: බන්ධන ශක්තිය = (Zmp + Nmn - M)c²
- **එක් නියුක්ලියෝනයකට බන්ධන ශක්තිය**: Fe-56 උපරිම (~8.8 MeV)

### 2. විකිරණශීලිතාව (Radioactivity)

- **α ක්ෂය**: He-2 න්‍යෂ්ටිය (⁴₂He) විමෝචනය, A අඩුවේ 4න්, Z අඩුවේ 2න්
- **β⁻ ක්ෂය**: නියුට්‍රෝනය ප්‍රෝටෝනයක් + ඉලෙක්ට්‍රෝනයක් + ප්‍රති-නියුට්‍රිනෝ, Z වැඩිවේ 1න්
- **β⁺ ක්ෂය**: ප්‍රෝටෝනය නියුට්‍රෝනයක් + පොසිට්‍රෝනයක් + නියුට්‍රිනෝ, Z අඩුවේ 1න්
- **γ විමෝචනය**: උද්වේගි න්‍යෂ්ටියෙන් ෆෝටෝන විමෝචනය (A, Z නොවෙනස්)
- **ක්ෂය නියමය**: N = N₀e^(-λt), T₁/₂ = ln2/λ = 0.693/λ
- **අර්ධ ආයු කාලය (Half-life)**: මුල් න්‍යෂ්ටි ප්‍රමාණයෙන් අඩක් ක්ෂය වීමට ගතවන කාලය
- **සක්‍රියතාව (Activity)**: A = λN = A₀e^(-λt)
- **කාබන්-14 කාල නිර්ණය**: t = (T₁/₂/ln2) × ln(N₀/N)

### 3. න්‍යෂ්ටික විඛණ්ඩනය (Nuclear Fission)

- **U-235 විඛණ්ඩනය**: ²³⁵U + n → ¹⁴¹Ba + ⁹²Kr + 2-3 n + ශක්තිය (~200 MeV)
- **දාම ප්‍රතික්‍රියාව**: නියුට්‍රෝන ගුණ කිරීම (critical mass)
- **න්‍යෂ්ටික බලාගාර**: න්‍යෂ්ටික විඛණ්ඩනයෙන් තාපය → වාෂ්ප → ටර්බයින → විදුලිය
- **න්‍යෂ්ටික ආයුධ**: පාලනය නොකළ දාම ප්‍රතික්‍රියාව

### 4. න්‍යෂ්ටික විලයනය (Nuclear Fusion)

- **හයිඩ්‍රජන් විලයනය**: ²H + ³H → ⁴He + n + 17.6 MeV
- **තාරකා ශක්ති ප්‍රභවය**: සූර්යයා තුළ ප්‍රෝටෝන-ප්‍රෝටෝන දාමය
- **Tokamak**: පාලනය කළ විලයනය සඳහා (ITER)
- **විලයන වාසි**: පිරිසිදු, ඉන්ධන බහුල, විකිරණ අඩු

### 5. ක්වොන්ටම් යාන්ත්‍ර විද්‍යාව (Quantum Mechanics)

- **Planck උපකල්පනය**: E = hf (h = 6.63×10⁻³⁴ Js)
- **තරංග-අංශු ද්වෛතවාදය**: de Broglie තරංග ආයාමය λ = h/p
- **අවිනිශ්චිතතා මූලධර්මය (Heisenberg)**: Δx·Δp ≥ ℏ/2
- **ශ්‍රෝඩින්ගර් සමීකරණය**: ක්වොන්ටම් තත්ත්ව විස්තර
- **උමං ආචරණය (Tunnelling)**: STM අන්වීක්ෂය
- **විද්‍යුත්-දුර්වල එක්සත් කිරීම**: Glashow-Weinberg-Salam
- **ප්‍රමිත ආකෘතිය**: ක්වාක් 6, ලෙප්ටෝන් 6, බෝසෝන

### 6. සාපේක්ෂතාව (Relativity)

- **විශේෂ සාපේක්ෂතාව (Einstein 1905)**:
  - ආලෝක ප්‍රවේගය සියලු නිරීක්ෂකයන්ට නියතයි
  - කාල ප්‍රසාරණය: t = γt₀, γ = 1/√(1-v²/c²)
  - දිග හැකිලීම: l = l₀/γ
  - E = mc²
- **සාමාන්‍ය සාපේක්ෂතාව**: ගුරුත්වාකර්ෂණය = අවකාශ-කාලයේ වක්‍රතාව

### 7. ප්‍රාථමික අංශු (Elementary Particles)

- **ලෙප්ටෝන්**: ඉලෙක්ට්‍රෝන (e), මුඔන් (μ), ටෝ (τ), නියුට්‍රිනෝ
- **ක්වාක්**: up, down, charm, strange, top, bottom (වර්ණ 3)
- **ගේජ් බෝසෝන**: ෆෝටෝනය (විද්‍යුත් චුම්බක), W⁺/W⁻/Z⁰ (දුර්වල), ග්ලූඕන (ප්‍රබල)
- **හිග්ස් බෝසෝනය**: ස්කන්ධයට හේතුව (CERN 2012)
